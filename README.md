# Repositorio de Prácticas de Pygame

Este repositorio contiene los ejercicios y mini-proyectos de pyagame2
Sesion4: Animación y Control de Tiempo
Crear animaciones fluidas, controlar FPS y simular físicas simples
sesion4_ej1.py: Círculo que rebota y acelera con cada rebote, controlado a 60 FPS.
sesion4_ej2.py`: Círculo que simula una pulsación, creciendo y encogiéndose
sesion4_ej3.py`: Simulación de gravedad donde una bola cae y rebota con pérdida de energía

Sesion5: Imágenes y Sprites
sesion5_ej1.py`: Carga una imagen y permite al usuario cambiar su tamaño con las teclas `+` y `-`
sesion5_ej2.py`: Carga 4 imágenes como frames para crear una animación de sprite
sesion5_mini.py`: Controla una nave que rota para apuntar al ratón y avanza al presionar `Espacio`

Sesion6: Colisiones y Aplicaciones
sesion6_ej1.py`: Un rectángulo de jugador que cambia a color verde al colisionar con un objetivo
sesion6_ej2.py`: Juego de "recolección" donde el jugador debe atrapar círculos que aparecen al azar, con un contador de puntos
sesion6_ej3.py`: Juego de "evitar" donde el jugador debe esquivar obstáculos móviles para no perder
sesion6_mini.py`: Mini-juego completo que combina movimiento de nave, recolección de puntos y evasión de obstáculos, con puntuación en pantalla
